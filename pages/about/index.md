## 你好，我是 autopoet

一名在前端领域不断探索、尝试与自我更新的开发者。

对未来的我来说，这里的一篇篇笔记也许并不复杂，但对现在的我而言，每一段文字都是我在那些“尚未胜任”的时刻，通过不断解决一个又一个问题所留下的阶梯。

### 我的技术栈与探索

- **前端深耕**：核心专注于 **Vue 3** 生态体系。深知 API 只是表象，所以更热衷于挖掘响应式原理、虚拟 DOM 与底层性能优化（如 `IntersectionObserver` 带来的零负担首屏缩减）。
- **工程化思维**：从习惯 Typora 全靠直觉的碎片化记录，过渡到基于 **SSG（静态站点生成）** 的结构化产出。在这个过程中，我学会了如何处理数据流的原子性操作、如何构建跨端边缘函数（Edge Functions）以及如何拦截那些隐蔽的安全威胁。
- **实践方向**：我倾向于通过造轮子来加深理解，在这个博客中，你可以看到我手动实现的 Vue 自定义指令、基于乐观更新（Optimistic UI）的交互组件，以及对 TypeScript 类型系统的应用。

### 核心信条

> "Every great developer you know got there by solving problems they were unqualified to solve."

这句话陪伴我度过了许多对代码逻辑一筹莫展的深夜。我深信，技术的边界不在于我已经掌握了什么，而在于我敢于面对哪些未知。
